## ✨ Features

- **Interactive Knowledge Graphs**: Visualize search results as connected nodes using our interactive *Rabbit Hole* mode.
- **Deep Categorization**: Search across unrefined theories, undocumented databases, hidden base internets, and more.
- **Node-Based Exploration**: Seamlessly ask follow-up questions right from individual search result nodes.
- **Modern Interface**: Clean, dark-mode optimized UI built with Tailwind CSS and React.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/anysearch.git
   cd anysearch
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Environment Variables:
   Rename `.env.example` to `.env` and add your required keys.

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Visualization**: React Flow (`@xyflow/react`)
- **Backend**: Node.js, Express
